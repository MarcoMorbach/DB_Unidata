# DB_Unidata

 
Este projeto foi desenvolvido inteiramente em MySQL.

Execute os arquivos na seguinte ordem em uma instância
MySQL limpa:
 
  1. ddl.sql         — Criação do banco e das tabelas
  2. dml.sql         — Inserção dos dados
  3. view.sql        — Criação da view vw_historico_academico
  4. trigger.sql     — Criação do trigger trg_calcula_situacao
  5. queries.sql     — Consultas Q1 a Q10
  6. transacoes.sql  — Cenários de transação (ACID)
 

EVIDÊNCIAS:
Os prints e resultados de cada script estão disponíveis
na pasta evidencias.
